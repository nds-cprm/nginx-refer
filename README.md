# nginix-refer
Instalação de referencia para servidor de nginx (https://www.nginx.org/), versao 1.18 em imagem linux debian:buster-slim, para container no formato docker.

## Especificações
Utiliza as portas 8080:80 e 8443:443

A configuração inicial, define: 
    usuario nginx acesso restrito, sem acesso a shell e sem diretorio,
